# Dana Vitan — Онлайн магазин

🌐 **[Виж сайта на живо → www.danavitan.com](https://www.danavitan.com)**

---

Онлайн магазин, изграден с **Next.js** и **Sanity CMS**. Поръчките се изпращат по имейл, а съдържанието на продуктите се управлява през Sanity Studio.

## Технологии

- **[Next.js 16](https://nextjs.org)** + **React 19** — frontend и сървърна логика
- **[Sanity CMS](https://www.sanity.io)** — управление на продукти и съдържание
- **[Tailwind CSS 4](https://tailwindcss.com)** + **styled-components** — стилизиране
- **[Framer Motion](https://www.framer.com/motion/)** — анимации
- **[React Hook Form](https://react-hook-form.com)** + **[Zod](https://zod.dev)** — форми и валидация
- **[Resend](https://resend.com)** — изпращане на поръчки по имейл

## Стартиране локално

```bash
npm run dev
```

Отвори [http://localhost:3000](http://localhost:3000) в браузъра.

## Полезни команди

```bash
npm run dev      # дев сървър
npm run build    # production build
npm run start    # стартира production build
npm run lint     # проверка на кода
npm run migrate  # миграция на продукти
npm run backfill # запълване на reда на бележките
```
